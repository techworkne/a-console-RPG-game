# Выживание

**Текстово-консольная RPG в мире постапокалипсиса.**

Вы — последний след прошлой цивилизации. В ваших руках — древний проклятый клинок, жаждущий душ, и верный, мощный, пистолет. Ваша цель — пройти через руины мертвых городов, полные мутантов и странных башен, чтобы добраться до легендарного Убежища. Выживание — ваша единственная награда.

---

## Особенности игры

*   **Динамичные бои:** Пошаговая боевая система с выбором атаки (ближний/дальний бой), использованием предметов и тактическим позиционированием.
*   **Проклятый артефакт:** Ваш клинок\пистолет может расти в силе при каждом приближение к убежищу.
*   **Система прогрессии:** Посещайте магазин и покупайте ресурсы или усиления.
*   **Исследование:** Находите заклинание в разрушенных башнях и используйте против своих врагов.

## Как начать играть

### Требования
*   Компилятор C++ с поддержкой стандарта C++17 (например, g++, clang++)
*   Терминал или командная строка

### Сборка и запуск (Linux/macOS)
```bash
# 1. Клонируйте репозиторий (или перейдите в папку с игрой)
git clone https://github.com/techworkne/a-console-RPG-game.git
cd a-console-RPG-game

# 2. Скомпилируйте проект
g++ -std=c++17 *.cpp -o wasteland_blade

# 3. Запустите игру
./wasteland_blade
